## üìä Tipos de Distribui√ß√µes

### Discretas vs. Cont√≠nuas

#### **Distribui√ß√µes Discretas**
Vari√°vel assume valores **cont√°veis** (inteiros).

**Fun√ß√£o Massa de Probabilidade (PMF):**
```
P(X = k) = probabilidade de X ser exatamente k
```

**Propriedades:**
```
‚àë P(X = k) = 1  (soma de todas as probabilidades = 1)

```

#### **Distribui√ß√µes Cont√≠nuas**
Vari√°vel assume valores em um **intervalo cont√≠nuo**.

**Fun√ß√£o Densidade de Probabilidade (PDF):**
```
f(x) = densidade em x
P(a ‚â§ X ‚â§ b) = ‚à´‚Çê·µá f(x) dx
```

**Propriedades:**
```
‚à´‚Çã‚àû^‚àû f(x) dx = 1  (√°rea total = 1)
P(X = x exato) = 0  (probabilidade de ponto exato √© zero!)
```

As f√≥rmulas **derivadas** s√£o diferentes porque cada distribui√ß√£o tem **f(x) ou P(X=x) diferente**!

## üìö Cat√°logo de Distribui√ß√µes Importantes

### 1. Binomial(n, p)

**Tipo:** Discreta

**Contexto:** n tentativas independentes, cada com probabilidade p de sucesso.

**PMF:**
```
P(X = k) = C(n,k) √ó p^k √ó (1-p)^(n-k)
```

**Momentos:**
```
M√©dia: Œº = np
Vari√¢ncia: œÉ¬≤ = np(1-p)
```

**Exemplo [[Aplica√ß√£o_ao_IoT_IDS|IDS]]:**
```python

dist = stats.binom(n=100, p=0.9)

# P(acertar exatamente 90)
print(dist.pmf(90))  # 0.132

# P(acertar 85 ou mais)
print(1 - dist.cdf(84))  # 0.942
```

### 2. Normal(Œº, œÉ¬≤)

**Tipo:** Cont√≠nua

**Contexto:** Distribui√ß√£o mais comum na natureza. Teorema Central do Limite.

**PDF:**
```
f(x) = (1/‚àö(2œÄœÉ¬≤)) √ó exp[-(x-Œº)¬≤/(2œÉ¬≤)]
```

**Momentos:**
```
M√©dia: Œº (par√¢metro)
Vari√¢ncia: œÉ¬≤ (par√¢metro)
```

**Propriedades:**
- Sim√©trica em torno de Œº
- Regra 68-95-99.7 (veja [[M√©dia_Desvio_Padr√£o_Erro_Padr√£o#Interpreta√ß√£o com Normal|regra emp√≠rica]])

**Exemplo:**
```python

# P(acur√°cia > 0.85)

# IC 95%

```

**Problema:** N√£o respeita limites [0,1] para propor√ß√µes! Veja [[The_Balanced_Accuracy_and_Its_Posterior_Distribution#Problema 1]].

### 3. [[Distribui√ß√£o_Beta|Beta(Œ±, Œ≤)]]

**Tipo:** Cont√≠nua

**Contexto:** Propor√ß√µes, probabilidades, acur√°cias. Prior conjugado da Binomial.

**PDF:**

**Momentos:**
```

Vari√¢ncia: œÉ¬≤ = Œ±Œ≤/[(Œ±+Œ≤)¬≤(Œ±+Œ≤+1)]
```

**Por que perfeita para [[Acur√°cia]]:**
- ‚úÖ Suporte [0,1] - respeita limites naturais!
- ‚úÖ Flex√≠vel - v√°rias formas com Œ±, Œ≤
- ‚úÖ Conjugada - matem√°tica elegante

Veja p√°gina dedicada: [[Distribui√ß√£o_Beta]]

### 4. Poisson(Œª)

**Tipo:** Discreta

**Contexto:** Contagem de eventos raros em intervalo fixo.

**PMF:**
```
P(X = k) = (Œª^k √ó e^(-Œª)) / k!
```

**Momentos:**
```

```

**Exemplo IDS:**

```python

```

### 5. Uniforme(a, b)

**Tipo:** Cont√≠nua

**Contexto:** Total ignor√¢ncia - todos os valores igualmente prov√°veis.

**PDF:**
```

```

**Momentos:**
```

```

**Uso em [[Infer√™ncia_Bayesiana]]:**

### 6. Exponencial(Œª)

**Tipo:** Cont√≠nua

**Contexto:** Tempo at√© o pr√≥ximo evento (processos de Poisson).

**PDF:**

**Momentos:**
```

Vari√¢ncia: 1/Œª¬≤
```

**Exemplo IDS:**

## üîÑ Rela√ß√µes Entre Distribui√ß√µes

### Fam√≠lias Conjugadas

**Binomial ‚Üî Beta**
```
Likelihood: Binomial(n, Œ∏)
Prior: Beta(Œ±, Œ≤)
‚Üí Posterior: Beta(Œ± + k, Œ≤ + n - k)
```

Usado no [[The_Balanced_Accuracy_and_Its_Posterior_Distribution|artigo]]!

**Poisson ‚Üî Gamma**
```
Likelihood: Poisson(Œª)
Prior: Gamma(Œ±, Œ≤)
‚Üí Posterior: Gamma(Œ± + ‚àëk, Œ≤ + n)
```

Veja [[Infer√™ncia_Bayesiana#Conjugate Priors|priors conjugados]].

### Aproxima√ß√µes

**Binomial ‚Üí Normal** (para n grande):
```
Binomial(n, p) ‚âà Normal(np, np(1-p))
```

**Poisson ‚Üí Normal** (para Œª grande):
```
Poisson(Œª) ‚âà Normal(Œª, Œª)
```

**Beta ‚Üí Normal** (para Œ±, Œ≤ grandes):
```
Beta(Œ±, Œ≤) ‚âà Normal(Œ±/(Œ±+Œ≤), ...)
```

## üßÆ Escolhendo a Distribui√ß√£o Certa

### Perguntas a Fazer

1. **Tipo de vari√°vel?**
   - Discreta ‚Üí Binomial, Poisson, etc.
   - Cont√≠nua ‚Üí Normal, Beta, etc.

2. **Suporte (valores poss√≠veis)?**
   - [0,1] ‚Üí Beta ‚úÖ
   - [0,‚àû) ‚Üí Exponencial, Gamma
   - (-‚àû,‚àû) ‚Üí Normal

3. **Processo gerador?**
   - n tentativas independentes ‚Üí Binomial
   - Contagem em intervalo ‚Üí Poisson
   - Propor√ß√£o/probabilidade ‚Üí Beta

4. **Propriedades desejadas?**
   - Sim√©trico ‚Üí Normal
   - Assim√©trico ‚Üí Beta (com Œ±‚â†Œ≤), Gamma

### Para [[Acur√°cia]]/[[Acur√°cia_Balanceada]]

**Escolha: Beta** ‚úÖ

**Raz√£o:**
1. Suporte [0,1] - respeita limites naturalmente
2. Flex√≠vel (m√∫ltiplas formas)
3. Conjugada √† Binomial (classifica√ß√£o √© Bernoulli)
4. Interpreta√ß√£o clara dos par√¢metros

**N√ÉO use Normal** para propor√ß√µes (pode violar [0,1])!

## üé≤ Simula√ß√£o e Visualiza√ß√£o

### Comparando Distribui√ß√µes

```python

plt.title('Beta(91, 11)')
plt.xlabel('Acur√°cia')

# Normal (problema!)
plt.subplot(133)
plt.plot(x_continuous, normal.pdf(x_continuous))

plt.title('Normal(0.9, 0.03¬≤) - Problema!')
plt.xlabel('Acur√°cia')
plt.legend()

plt.tight_layout()
plt.show()
```

### Amostragem
